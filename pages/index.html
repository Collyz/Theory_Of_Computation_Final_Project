Finite Automata Designer - ToC Final Project

How to use:
  • Create State: Double click inside box
  • Create Accept State: Double click existing state
  • Label States: Right click existing state
  • Start State: Label state as q_0
  • Create Arrows: Click previous state first, THEN hold shift and click next state
  • Delete States: Click on state once, (make sure border is red), press 'Delete' key WIP
